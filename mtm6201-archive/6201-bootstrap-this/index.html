About (Parts of) Bootstrap

Grid

Bootstrap Grid is like CSS Flex, but it's so much easier to use (especially when you're building responsive sites)! You just add classes to specify how you want your content to look using classes like "sm" "md" and "lg" for viewport widths and then how much of the screen you want that element to take up and BAM! It looks good! Who wouldn't want to use that?

Pill Badges

Like This!

They're super easy to do but I do really like the way they look. There's not much to talk about though (it's just giving your span a class of badge then styling it with the same color class names as the rest of Bootstrap to look cool).

Carousels

To be honest, I don't have ANY idea how I would go about coding this so I think Bootstrap will be an asset going forward. I'm assuming there's JS involved but I can't really make sense of it. I chose to have captions on my carousel as I want to use it in my portfolio site, but it's super easy to hide captions on smaller screens by, once again, just adding a class (in this case the class is d-none, which is the bootsrap version of CSS' display: none), and then a d-md-block (display: block at medium viewport width).
